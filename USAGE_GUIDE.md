### Step 1: Get Your OpenAI API Key
1. Visit [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Sign up or log in to your OpenAI account
3. Click "Create new secret key"
4. Copy your API key (starts with `sk-`)
5. **Important**: Keep this key secure and never share it publicly

### Step 2: Start the Application
```bash
npm run dev
```
The app will open at `http://localhost:5173`

### Step 3: Configure Settings
1. When you first open the app, a settings modal will appear
2. Paste your OpenAI API key
3. Choose your model:
   - **GPT-4o** (Recommended) - Latest and most capable
   - **GPT-4o Mini** - Faster and more cost-effective
   - **GPT-4 Turbo** - Advanced reasoning
   - **GPT-3.5 Turbo** - Budget-friendly option
4. Adjust temperature (0.7 is recommended for interviews)
5. Click "Save Settings"
